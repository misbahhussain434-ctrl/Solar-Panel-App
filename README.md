# Solar-Panel-App

The Solar Panel Recommendation App is a smart application that helps users estimate the number of solar panels required, system size, and estimated energy production based on their electricity usage.
This application helps homeowners and businesses understand how they can switch to renewable energy and reduce electricity costs.

**Features**

Calculate required solar panel system size
Estimate number of solar panels needed
Estimate monthly energy production
Simple and user-friendly interface
Helps users understand solar energy potential
